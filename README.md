## Develop

### Assets
Use ```src/assets``` folder for all media assets like images or videos.
All those will copy into ```dist/assets``` folder without any changes.

## Commands

### Dev server
For start dev server use
```
> npm run serve
```
### Tests
For start lint tests use
```
> npm run test
```
### Build
For build use
```
> npm run build
```
Before deploy use
```
> npm run build:prod
```
